Click the *Start* button to set up the sandbox environment for this training, this may take up to 1 min 30 sec. Once the environment is ready, click the *Start* button again.

- You'll need both an **OpenAI** API key and a **Tavily** API key to complete this track.
  - OpenAI: sign up at [platform.openai.com](https://platform.openai.com/signup).
  - Tavily: sign up at [tavily.com](https://tavily.com) — the free tier covers 1000 searches/month, more than enough for this track.
  - We recommend getting both before you start.
- There are 3 challenges. If your session is idle for more than 10 minutes the session will stop and you'll need to restart the track and re-enter your API keys.
- Tracks can be started up to 5 times. You can select which challenge to run via the *Progress* button in the top right.
- This track assumes you've completed the **Dapr Agents** fundamentals track (or are comfortable with `DurableAgent`, tool calls, and basic Dapr Workflow concepts).
