The sandbox for this challenge is being prepared. Once it's ready, click the *Start* button.

In this challenge, you'll:

- Scaffold a new Aspire solution from the `aspire-starter` template.
- Configure the AppHost launch URLs so the Aspire dashboard is reachable in the sandbox environment.
- Add the NuGet packages required for Dapr Workflow.

If you have any questions or feedback about this track, you can let us know in the *#general* channel of the [Dapr Discord server](https://diagrid.ws/dapr-discord).
